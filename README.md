# Data Warehouse & SQL Analytics Pipeline

## Overview
This project demonstrates an end-to-end data engineering pipeline using Python and SQL.

## Workflow
- Created sales dataset
- Performed data transformation (revenue calculation)
- Loaded data into SQLite database
- Executed SQL queries for business insights

## Tech Stack
Python, Pandas, SQLite

## Insights
- Total revenue
- Revenue by product
- Top customers

## How to Run
pip install pandas
python warehouse.py
